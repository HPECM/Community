Overview

This add-in allows you to:

  • register an individual email in Content Manager,
  • register the attachments on an email in Content Manager,
  • move one or more emails to a linked folder causing them to be filed in Content Manager,
  • create Checkin Styles which contain pre=set a particular Record Type and default values for various Record properties,
  • create a linked folder,
  • set the default Record Type
  • pin the Record task pane to avoid having to access the menu repeatedly when you have multiple emails to file or view in Content Manager.
  • view and edit the Record properties once an email is filed in Content Manager, and
  • easily navigate to other Records related to this Record from the context tab.

The Content Manager menu

The menu for the Content Manager integration is found in the ribbon in desktop Outlook but in the 'More actions' menu within a mail item in Outlook Online. Throughout this documentation it will be referred to as 'the menu'.

File a single email

To file a single email, or attachments from a single email:

  • select the email
  • from the menu select 'Record'
  • select a Record Type or Check in Style, then
  • enter the details and click 'Register in Content Manager.

If there are attachments on the email the first screen, instead of showing the Record Type quick select, will allow you to choose one or more attachments and/or the email itself. At this point you may also choose to bypass the data entry form.

Create a Check in Style

A Check in Style is simply a way to set up default values in Record properties. For example if you have fields that always contain the same value whenever you file email from a particular customer you might create a Check in Style to save re-entering those values every time.To create a Check in Style:

  1. from the menu select 'Linked Folders',
  2. from the context menu in the linked folder list select the 'New Check in Style' icon,
  3. select a Record Type and enter default values in fields, then
  4. select 'Register in Content Manager'.

Create a linked folder

A linked folder is a special type of Check in Style that has a link to one of the folders in your mail box. This allows the server side component of the integration to monitor these linked folders and register any mail item that is moved to the folder according to the settings in the Check in Style. Because the email is filed by a server process linked folders do not allow you to access the data entry form when you move the item to the folder. To create a linked folder:

  1. from the menu select 'Linked Folders',
  2. from the context menu in the linked folder list select the 'New Linked Folder' icon,
  3. select either a pre-existing folder or choose 'Auto Create New Folder' to have the integration create a folder for you,
  4. select a Record Type and fill in any properties you want to be used as defaults on items filed using this folder
  5. the 'Processing' tab contains properties that determine how this folder behaves, does it file only attachments on emails and does it delete mail items from Outlook as the are filed to Content Manager.

Record Context

Once a document has been registered in Content Manager the the Task Pane should open automatically whenever the document is opening. This Task Pane contains a Context tab with quick links to those Records that are:

  • in the same container
  • have the same contacts, or
  • are related to.

The context pane may also be used to do a search for any Record in Content Manager.

User Options

The User Options menu item allows you to set the default Record Type.

User Interface Components

Quick Select

Fields in the Record entry form that contain Content Manager objects, such as Container or Author, are populated using quick select fields. These quick select fields are simpler than the corresponding fields in either the desktop or web clients, you may find an object by:

  • clicking the magnifying glass than selecting one of the shortcut icons to the left of the of the list
  • typing in the field to do a simple search (by name or title)
  • clicking the spanner icon at the left of the field to toggle into advanced search which allows you to use the Content Manager search language in the field, for example `contains:all' to find a Records that contain at least one other Record.

Once you have found the appropriate object in the list select it to populate the field.

Record Entry form tabs

The tabs used in the record entry form have to fit into the width of the task pane, for this reason they have an overflow feature. If there is more than one tab in the Record Entry form and it is likely the combined size of the tabs will be larger than the available width then overflowing tabs will be shown not as tabs but in the overflow menu, represented by the an ellipsis menu.

Properties Pane

Records and Check in Places are displayed using the properties pane, this is the pane that is displayed in the task pane when you select the 'Record' menu item for an item that exists in Content Manager. The properties pane shows the list of properties you have selected to show in the View Pane. To remove a propery from display use the X icon to the right of the property value. To add a propert use the combobox at the bottom of the pane to choose the property then select 'Add'. To reset the displayed properties to the default choose 'Get default properties andf fields for view pane' from the context menu.
