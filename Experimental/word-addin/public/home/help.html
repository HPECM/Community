Help

Overview

This add-in allows you to:

  • register a document in CM by creating a Record linked to this document,
  • check the document into the linked Record,
  • request deletion of the document in OneDrive,
  • finalise the Record,
  • link to the Record in the Content Manager Web Client,
  • favorite the record,
  • view the record properties,
  • request sharing of a Record already checked out by another user, and
  • open a record from Content Manager.

Registering a document

To register a document into Content Manager:
  1. from the Content Manager ribbon tab select 'Record',
  2. from the Task Pane choose a Record Type,
  3. supply the required properties on the Record, then
  4. click the 'Register in Content Manager' button.

Support in OneDrive local and web

The integration should work seamlessly no matter how the file is opened from OneDrive, including:

  • OneDrive Web Site,
  • locally synced OneDrive files,
  • open from OneDrive in desktop Office application, or
  • document stored in SharePoint document library

Office support

The only Office applications supported by this integration are:

  • Word,
  • Excel, and
  • PowerPoint.

Viewing the Record properties

Once the document has been filed in Content Manager the task pane should open automatically to display the list of properties available in the Content Manager view pane. If this task pane is not open it may be opened using the same 'Record' menu option as was used to originally register them document.

Transferring a document to Content Manager

Once the document has been registered in Content Manager its content may be updated in one of two ways:

  • manually, by selecting the 'Check in an Electronic Document' option in the Task Pane menu, or
  • automatically on close by selecting the 'Enable check in and delete on close' option from the Task Pane menu.

Re-opening a document from Content Manager

If you have used the 'Check in and delete on close' method to update a Record in Content Manager then a link to that Record will have been created for you in OneDrive. This link may be used to re-open the Record. Opening the record will cause it to be checked out to you and a copy of the document placed in your 'Content Manager Documents' folder in OneDrive.

Previewing a document from Content Manager

If you have used the 'Check in and delete on close' method to update a Record in Content Manager then a link to that Record will have been created for you in OneDrive. This link may be used to preview the document within Content Manager. The ellipsis menu on the document contains a 'preview' option, this will open a PDF rendition of the document from within Content Manager for preview. From this preview you may to choose to open the actual document which will cause it to be copied back into your 'Content Manager Documents' folder in OneDrive.

Sharing a Record

The integration will prompt you to send a request to share a Record if you use the open dialog to find a Record in Content Manager that is already checked out to someone elses OneDrive. If the owner of the checked out Record allows sharing you will be able to edit the instance in their OneDrive but only they will have permission to check the document into Content Manager.

Paste Record Details

The context menu in the task pane contains a 'paste' item which allows you to paste the:

  • Record Title,
  • Record Number, or
  • a link to the Record in the Web Client.

These paste menu is availeble both in the context menu for the current Record and also in the list of Records displayed inside the Context tab.

Record Context

Once a document has been registered in Content Manager the the Task Pane should open automatically whenever the document is opening. This Task Pane contains a Context tab with quick links to those Records that are:

  • in the same container,
  • have the same contacts, or
  • are related to.

The context pane may also be used to do a search for any Record in Content Manager.

User Interface Components

Quick Select

Fields in the Record entry form that contain Content Manager objects, such as Container or Author, are populated using quick select fields. These quick select fields are simpler than the corresponding fields in either the desktop or web clients, you may find an object by:

  • clicking the magnifying glass than selecting one of the shortcut icons to the left of the of the list
  • typing in the field to do a simple search (by name or title)
  • clicking the spanner icon at the left of the field to toggle into advanced search which allows you to use the Content Manager search language in the field, for example `contains:all' to find a Records that contain at least one other Record.

Once you have found the appropriate object in the list select it to populate the field.

Open dialog

The open dialog allows you to search all Records in CM, even non electronic ones. To use it:

  1. enter a search term (or select a shortcut from the left)
  2. wait for the result list to populate
  3. select a Record
  4. click OK
  5. if the Document already exists in O365 it will be opened, if not it will be copied to the folder 'Content Manager Documents' from CM and then opened.

Record Entry form tabs

The tabs used in the record entry form have to fit into the width of the task pane, for this reason they have an overflow feature. If there is more than one tab in the Record Entry form and it is likely the combined size of the tabs will be larger than the available width then overflowing tabs will be shown not as tabs but in the overflow menu, represented by the an ellipsis menu.

Properties Pane

Records and Check in Places are displayed using the properties pane, this is the pane that is displayed in the task pane when you select the 'Record' menu item for an item that exists in Content Manager. The properties pane shows the list of properties you have selected to show in the View Pane. To remove a property from display use the X icon to the right of the property value. To add a property use the combobox at the bottom of the pane to choose the property then select 'Add'. To reset the displayed properties to the default choose 'Get default properties and fields for view pane' from the context menu.
